## 🚀 The Vision
Every day, hostels waste incredible amounts of food simply because they don't know exactly how many students will be eating. **Smart Mess** solves this by providing a gorgeous, seamless interface for students to confirm their meals, giving mess staff the exact data they need to cook the right amount.

## ✨ Core Features

| Feature | Description |
| :--- | :--- |
| **📱 Student Dashboard** | A highly responsive, mobile-first interface for students to easily confirm or skip upcoming meals with a single tap. |
| **💻 Admin Portal** | A powerful desktop dashboard for mess staff to track the exact required food quantities based on real-time data. |
| **📊 Smart Analytics** | Visual data insights, progress bars, and tracking of both food waste reduction and cost savings over time. |
| **🎨 Premium UI/UX** | Built with an absolutely stunning glassmorphism design, fluid CSS animations, and a sleek modern dark mode. |

---

## 🛠️ Tech Stack 

- **Frontend Framework:** React 18 + Vite ⚡
- **Design System:** Custom Modern CSS (Glassmorphism, CSS Variables, Deep Dark Theme) 💎
- **State Management:** React Context API
